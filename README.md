# Azure FinOps Cost Optimization Automation

## Goal

Build a FinOps automation project that tracks, reports, and optimizes Azure cloud spending.

## Azure Services

- Azure Cost Management
- Azure Budgets
- Azure Advisor
- Azure Resource Graph
- Azure Automation
- Log Analytics Workspace

## DevOps Skills Demonstrated

- Cloud cost governance
- Budget automation
- Tag-based reporting
- Idle resource detection
- Advisor recommendation exports
- Scheduled optimization workflows

## Example Automations

```text
Find unattached disks
Find idle public IPs
Find oversized VMs
Report untagged resources
Create monthly cost report
Send budget alerts
```
